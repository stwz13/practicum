# PlayStation Games Info 2/15/2025 #
### Содержание столбцов датасета: ###

Датасет PlayStation Games Info 2/15/2025 содержит следующие столбцы:

1. ***game_name***
>Содержит названия игр.
2. ***highest_price***
>Самая высокая зарегистрированная цена (в PlayStation Store, в евро).
3. ***release_date***
>Дата выхода игры на указанной платформе PlayStation.
4. ***genre***
>Первичные и вторичные жанры.
5. ***publisher***
>Издательство или студия, ответственная за выпуск.
6. ***platform***
>Платформа PlayStation (PS3, PS4, PS5 и т.д.).
7. ***metacritic_store***
>Средний рейтинг критиков (0–100) по версии Metacritic.
8. ***metacritic_rating_count***
>Количество рецензий критиков, учтенных на Metacritic.
9. ***metacritic_user_score***
>Средняя оценка пользователей (0–10) от Metacritic.
10. ***metacritic_user_rating_count***
>Количество пользовательских оценок, учитываемых на Metacritic.
11. ***playstation_score***
>Оценка пользователей PlayStation Store (0–5).
12. ***playstation_rating_count***
>Общее количество оценок пользователей PlayStation Store.
13. ***"#"***
>Пустой столбец
